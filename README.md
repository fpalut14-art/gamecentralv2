# GameCentral Production Base

Next.js App Router + Firebase Auth + Firestore tabanlı gamer marketplace altyapısı.

## Modüller
- Auth: login/register
- Role: admin/seller/user
- Marketplace: products, listing detail, orders
- Admin: dashboard, users, seller requests, listings, orders, ads, reports, support, logs
- Seller: seller dashboard
- Ads: ad request + admin moderation
- Notifications
- Buyer-seller chat
- Live support tickets
- Reports/complaints

## Çalıştırma
```bash
npm install
npm run dev
```
